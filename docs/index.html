Notion Daily Quote?

Notion Daily Quote는 notion으로 일기를 작성하는 분들이 매일의 인용구를 직접 타이핑하지 않아도 되도록 대신 입력해 드리는 서비스입니다.

Notion Daily Quote 사용 방법

  1. Notion > Integration > 통합 추가 선택
  2. 반복 생성 템플릿 내에 [QUOTE] 삽입
  3. 데이터베이스 링크를 확보하여 하단에 입력 후 등록
  4. 매일 01시에 [QUOTE]가 그날의 인용구로 갱신됩니다

현재는 법구경만 지원하고 있습니다. 데이터가 확보되는대로 다른 인용구도 지원하겠습니다
